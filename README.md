# ✋ Hand Gesture Control（開発中）

> カメラを使って手のジェスチャーでPCを操作するシステム

---

## 📌 概要
本プロジェクトは手のジェスチャーを認識し、  
マウスとキーボードの使用を最小限に抑えた  
**より便利にPC操作**を実現することを目指しています。

---

## 🚀 主な機能
- ✋ ジェスチャーを利用した音量·明るさ調整  
- 🖱 ジェスチャーを利用したクリック動作
- 🎯 リアルタイムハンドトラッキング  
- 📏 指の距離·角度による制御  
- 📺 今後: YouTube·カカオトークの操作支援を予定  

---
